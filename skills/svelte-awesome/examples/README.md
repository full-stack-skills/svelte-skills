# Svelte Examples

本目录提供 Svelte 5 的离线可执行代码示例，配合 `svelte-awesome` 入口技能使用。

## 文件说明

| 文件 | 内容 |
|------|------|
| `getting-started.md` | 项目创建、首个组件、工具链配置 |
| `component-patterns.md` | 常用组件模式、Snippet 复用、事件处理 |
| `project-structure.md` | Svelte + SvelteKit 项目结构、路由组织 |

## 使用建议

- 需要快速复制代码片段时，直接从对应示例文件中查找
- 需要理解完整项目结构时，参考 `project-structure.md`
- 需要运行时深入细节时，切换到对应子技能的 `references/` 目录
