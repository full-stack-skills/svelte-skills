# Svelte Lifecycle References

本目录提供 Svelte 5 生命周期、Stores、Context、Testing、浏览器支持的参考文档。

## 文件说明

| 文件 | 内容 |
|---|---|
| `lifecycle-hooks-reference.md` | onMount/onDestroy/tick 完整签名、SSR 注意事项、迁移指南 |
| `stores-api-reference.md` | svelte/store 完整 API、TypeScript 类型、Store 契约 |
| `context-api-reference.md` | createContext/setContext/getContext/hasContext、SSR 行为 |
| `testing-reference.md` | Vitest/Storybook/Playwright 完整 setup 指南 |
| `browser-support.md` | 浏览器最低版本、例外特性、polyfill 指南 |

## 使用建议

- 需要查 API 签名时查阅对应文件
- 与 `../SKILL.md` 的 Critical 章节、`../examples/` 的代码示例配合使用
