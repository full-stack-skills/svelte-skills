# Svelte Styling Examples

This directory contains practical examples for Svelte 5 styling patterns.

## Example Files

| File | Description |
|------|-------------|
| [scoped-styles.md](./scoped-styles.md) | Scoped styles, :global(), CSS custom properties, class directive, specificity, when styles don't apply, compiled output |
| [scoped-keyframes.md](./scoped-keyframes.md) | `@keyframes` scoping, the `-global-` prefix, naming, and triggering animations from JS |
| [global-patterns.md](./global-patterns.md) | `:global(...)` single, nested, block, and how to reach into slots |
| [style-directive.md](./style-directive.md) | `style:property` shorthand, expressions, `style:--var`, `\|important`, and `style:` vs `style=""` precedence |
| [nested-style.md](./nested-style.md) | Raw `<style>` tags inside templates, `{#if}`/`{#each}` style fragments, lifecycle |
| [tailwind.md](./tailwind.md) | Tailwind CSS integration, arbitrary values, dynamic theming, safelisting, twMerge |

## Quick Navigation

- **Scoped styles** — start with [scoped-styles.md](./scoped-styles.md)
- **Keyframes** — see [scoped-keyframes.md](./scoped-keyframes.md) for the `-global-` prefix and naming
- **Global selectors** — see [global-patterns.md](./global-patterns.md) for `:global(...)` forms
- **`style:` directive** — see [style-directive.md](./style-directive.md) for shorthand and `style:--var`
- **Nested `<style>`** — see [nested-style.md](./nested-style.md) for raw, unscoped style tags
- **Tailwind** — see [tailwind.md](./tailwind.md) for utility-class integration
