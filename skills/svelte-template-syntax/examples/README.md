# Svelte Template Syntax Examples

本目录提供 Svelte 模板语法的离线可执行代码示例。

## 文件说明

| 文件 | 内容 |
|------|------|
| `if-await-snippet.md` | {#if}/{#each}/{#await}/{#snippet} 基础到高级用法 |
| `bind-directives.md` | bind:value/checked/group/object 等所有绑定指令 |
| `event-handlers.md` | onclick vs on:click、事件委托、事件对象处理 |
| `render-tags.md` | {@render}/{@html}/{@debug}/{@const} 特殊渲染标签 |

## 使用建议

- 需要复制模板代码时，直接从对应文件获取
- 需要理解语义差异时，结合 `references/` 目录的理论说明
- 完整模板语法参考见 `../SKILL.md`
