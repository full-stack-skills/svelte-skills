# `sv` examples

Concrete, copy-paste-ready invocations of every `sv` subcommand.

| File | Covers |
| ---- | ------ |
| `sv-create.md` | `sv create` — every CLI option and common workflows |
| `sv-add.md` | `sv add` — each official add-on (drizzle, tailwind, prettier, eslint, mdsvex, vitest, playwright, storybook, paraglide, mcp, better-auth, sveltekit-adapter, experimental) |
| `sv-check.md` | `sv check` — command variants, options, watch mode, machine-readable output |
| `sv-migrate.md` | `sv migrate` — each migration (svelte-5, sveltekit-2, app-state, self-closing-tags, package, routes, svelte-4) |
| `sv-utils.md` | `@sveltejs/sv-utils` — `defineAddon`, transforms, `svelteConfig.find`, `pnpm.allowBuilds` |

For the full option reference see `../references/`.
